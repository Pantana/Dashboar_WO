Cara Hubungkan Sheets

  1. Buka file Google Sheets Anda.
  2. Pilih Ekstensi > Apps Script.
  3. Salin kode dari google-apps-script.js ke sana.
  4. Klik Terapkan > Penerapan Baru.
  5. Pilih jenis Aplikasi Web, akses oleh Siapa Saja (Anyone).
  6. Salin URL Web App dan paste di atas.
